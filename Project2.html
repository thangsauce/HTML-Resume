Dice Simulator

Project Description

  • Demonstrates basic object-oriented programming concepts in Java, including encapsulation, constructors, getters/setters, and use of the Random class for simulation
  • Simulates dice rolling using a Die class that initializes, rolls a six-sided die with random values (1–6), tracks the current face value, and supports a larger program that performs 1,000 rolls to build a histogram of outcome distributions

Skills Learned

  • Object-oriented programming principles in Java
  • Use Random class to simulate real-world randomness
  • Individual classes work together in a larger program
